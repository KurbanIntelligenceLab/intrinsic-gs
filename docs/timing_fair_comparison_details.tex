\section{Purpose}
This note documents the fair runtime comparison used for the paper.  The key point is that the small \texttt{spectral\_total\_s} number is only our graph-and-clustering component.  It should not be compared directly against TRASE's SAM-plus-feature pipeline.  The fair comparison below uses mask-production stages on both sides.

\section{Recommended Paper Wording}
A conservative, defensible wording is:

\begin{quote}
Starting from trained checkpoints and input frames, our mask-production stage takes 180.2 seconds per HyperNeRF scene on average, including graph construction, clustering, and cluster-ID rendering.  The corresponding TRASE-style SAM+feature mask-production pipeline takes 2259.6 seconds per scene on the ten-scene HyperNeRF set, giving a 12.5$\times$ speedup.
\end{quote}
